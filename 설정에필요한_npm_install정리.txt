STalkSecretServer 

STalkSecretServer 보다 한단계 상위디렉토리에서 입력
$ npm install -g express-generator
$ express STalkSecretServer		
	y (express용 프로젝트를 생성함 이미 디렉토리가 있으면 디렉토리 안에 기본파일들 생성)
		*이미 기본파일이 있고, 수정된 상태라면 해당 명령은 수행하지 말 것!
